# Cinderella 🔥

一度開いたら消える、ワンタイムの秘密共有。

パスワードや API キーを、**一度だけ開けるリンク**で相手に渡せます。
秘密はあなたのブラウザの中で暗号化され、**サーバーには中身が届きません**。
相手がリンクを開いた瞬間に表示され、そのまま消えます。

### 送る

1. 秘密を入力する（パスワード・API キー・メモなど）。
2. 有効期限（1 時間 / 24 時間 / 7 日）と、開ける回数（1 / 3 / 10 回）を選ぶ。
3. （任意）パスフレーズを設定し、**リンクとは別の経路**で相手に伝える。
4. 「Encrypt & Create link」を押すとリンクができる。コピーして相手へ送る。

### 受け取る

1. リンクを開く。
2. 「Reveal secret」を押すと、中身が**一度だけ**表示される。
3. 2 回目以降は "Already opened"。もし最初に開く前にこれが出たら、**誰かが先に開いた合図**なので送り主に連絡を。

## 安全について

- 暗号化も復号も**あなたのブラウザの中だけ**で行われます。復号の鍵はリンクの `#` より後ろに入っていて、サーバーには送られません。
- そのため、サーバーやログが漏れても**中身は読めません**。
- 一度開くと消えるので、リンクの使い回しや盗み見に強い設計です。
- とくに機微な秘密は、パスフレーズを別の経路で渡すと、リンクが漏れても開けなくなります。
